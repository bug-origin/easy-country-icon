# Security Policy

## Supported Versions

We release patches for security vulnerabilities for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within easy-country-icon, please send an email to [your-email@example.com]. All security vulnerabilities will be promptly addressed.

Please do not publicly disclose the issue until it has been addressed by the team.

### What to Include

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Suggested fix (if any)

### Response Time

- We will acknowledge receipt of your vulnerability report within 48 hours
- We will provide a more detailed response within 5 business days
- We will work on a fix and release it as soon as possible

## Security Best Practices

When using this package:

1. Always use the latest version
2. Review the CHANGELOG for security updates
3. Use npm audit to check for vulnerabilities in dependencies
4. Follow security best practices for your specific use case

Thank you for helping keep easy-country-icon and its users safe!
