## [1.0.0] - 2026-01-04

### Added
- Initial release of easy-country-icon
- Emoji-based country flag icons for 194 countries
- SVG support with 250+ high-quality flag icons
- Tree-shaking support for optimal bundle size
- TypeScript support with full type definitions
- Dual module format (ESM and CJS)
- On-demand import support
- Zero dependencies
- Country search functionality
- SVG generation utilities

### Features
- `getCountryEmoji()` - Get flag emoji by country code
- `getCountryData()` - Get complete country information
- `getAllCountries()` - Get all countries data
- `searchCountries()` - Search countries by name or code
- `getCountrySVG()` - Generate SVG from emoji
- `getCountryStyledSVG()` - Generate styled SVG with background
- Direct SVG file imports from package
